# Dating Advice Imitation Game

## Project Overview
This project is a Reddit post comparison website that allows users to compare different posts based on various metrics. It aims to provide insights and facilitate discussions on the effectiveness of various types of advice given in the context of dating and relationships.

## Features
- Compare Reddit posts side by side.
- User-friendly interface for easy navigation.
- Sorting options to filter the posts based on popularity, date, or relevance.
- Option to share comparisons with others.
- Responsive design for access on mobile and desktop.

## Setup Instructions
To get started with the Dating Advice Imitation Game, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/casusscribere/dating-advice-imitation-game.git
   cd dating-advice-imitation-game
   ```

2. **Install dependencies**:
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the application**:
   Start the local server:
   ```bash
   npm start
   ```
   The app will be running at `http://localhost:3000`.

4. **Build for production**:
   For a production build, run:
   ```bash
   npm run build
   ```

5. **Access the app**: Open your browser and go to `http://localhost:3000` to start using the application.
